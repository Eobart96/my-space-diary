# My Space - Diary & Nutrition App

Production-ready web application with diary and nutrition tracking modules.

## Technology Stack

- **Backend**: Node.js + Express + TypeScript + PostgreSQL
- **Frontend**: React + TypeScript + TailwindCSS
- **Authentication**: JWT
- **Database**: PostgreSQL
- **Deployment**: Docker + docker-compose

## Quick Start

```bash
# Clone and run
docker-compose up -d
```

Access at http://localhost:3000

## Project Structure

```
├── backend/          # Node.js API server
├── frontend/         # React application
├── docker-compose.yml
└── README.md
```

## Features

- **Diary Module**: CRUD diary entries with mood tracking
- **Nutrition Module**: Track meals with calories and macronutrients
- **Authentication**: Simple email/password login with JWT
- **Responsive Design**: Works on desktop and mobile
