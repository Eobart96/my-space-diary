MySpace Diary - Windows Release
================================

Запуск:
1. Дважды кликните на start-windows.bat
2. Откройте браузер и перейдите на http://localhost:5173
3. Наслаждайтесь синхронизацией!

Файлы:
- myspace-sync-windows.exe - Go sync engine
- start-windows.bat - Скрипт запуска
- web/ - React приложение

Порты:
- 8080 - Go sync engine API
- 5173 - React веб-приложение

Требования:
- Windows 10/11
- Никаких дополнительных зависимостей!
